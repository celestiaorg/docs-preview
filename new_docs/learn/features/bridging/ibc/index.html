Summary

IBC (Inter-Blockchain Communication)Â  is a protocol for authenticated, permissionless message passing between blockchains. Along with Hyperlane, it is one of the standards used on Celestia for cross-chain asset transfers and messages.

You can use IBC to bridge assets between Celestia-native chains along with other chains.

Celestia uses relayers to forward packets between chains. Relayers are processes that can be run by anyone which constantly scan for outbound packets on one chain and submits these packets alongside corresponding proofs on the destination chain.
